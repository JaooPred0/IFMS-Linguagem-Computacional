Operadores aritméticos

Operadores aritméticos em JavaScript são usados ​​para realizar cálculos matemáticos. Aqui estão os mais comumente usados:

  • Adição (+): Adiciona dois valores.

    • let a = 5;

    let b = 3;

    let soma = a + b;

  • Subtração (-): Subtrai um valor de outro.

    • let a = 10;

    let b = 5;

    let diferente = a - b;

  • Multiplicação (*:): Multiplica dois valores.

    • let a = 2;

    let b = 5;

    let produto = a * b;

  • Divisão (/): Divide um valor por outro.

    • let a = 10;

    let b = 2;

    let quociente = a / b;

  • Módulo (%): Retorna o restante de uma operação de divisão.

    • let a = 5;

    let b = 3;

    let restante = a % b;

  • Exponenciação ( ):** Eleva um valor à potência de outro valor.

    • let a = 2;

    let b = 4;

    let exponenciação = a ** b;

  • Incremento (++): Aumenta o valor de uma variável em 1.

    • let a = 5;

    a++;

  • Decremento (--): Diminui o valor de uma variável em 1.

    • let a = 5;

    a--;

    É importante observar que ao usar operadores aritméticos com valores não numéricos, o JavaScript tentará primeiro convertê-los em números. No entanto, isso às vezes pode levar a resultados inesperados, por isso geralmente é uma boa ideia certificar-se de que você está trabalhando explicitamente com números.
