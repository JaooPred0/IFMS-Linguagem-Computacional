Estruturas Condicionais Simples: IF ELSE

Em JavaScript, uma if instrução permite executar um bloco de código somente se uma condição especificada for verdadeira. A sintaxe básica de uma if instrução é:

if( condicao ) {

ㅤㅤ// código a ser executado se a condição for verdadeira

}

Aqui, condition pode ser qualquer expressão avaliada como um valor booleano (true ou false). Se a condição for avaliada como true, o código entre chaves ({}) será executado. Se a condição for avaliada como false, o código entre chaves será ignorado.

Aqui está um exemplo que demonstra como usar uma if instrução para verificar se um número é positivo ou negativo:

let num = -5;

if(num > 0 {

ㅤㅤconsole.log(num, 'é um número positivo.');

} else {

ㅤㅤconsole.log(num, 'é um número negativo.');

}

Estruturas Condicionais Compostas: IF ELSE

Neste exemplo, primeiro declaramos uma variável `num` e atribuímos a ela o valor -5. Em seguida, usamos uma instrução `if` para verificar se `num` é maior que 0. Como -5 não é maior que 0, a condição é avaliada como falsa e o código dentro do bloco `else` é executado. A saída deste código seria o resultado do bloco `else`.

As estruturas condicionais compostas em JavaScript permitem que um programa tome diferentes caminhos dependendo de diferentes condições serem atendidas. Existem dois tipos principais de estruturas condicionais compostas em JavaScript: `if...else if...else` e `switch...case`.

Se o valor da expressão corresponder a um dos valores de caso, o código desse caso será executado até que um break seja encontrado. Se nenhum caso corresponder ao valor da expressão, o código do bloco default será executado. O bloco default é opcional e é executado quando nenhum dos casos corresponde ao valor da expressão.
