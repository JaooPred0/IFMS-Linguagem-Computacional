For{}

O for loop é um tipo de loop em JavaScript que permite executar um bloco de código repetidamente enquanto uma determinada condição for verdadeira. A sintaxe básica de um for loop é a seguinte:

for (inicialização; condição; incremento) {

ㅤㅤ// Bloco de código a ser executado repetidamente

}

Nesta estrutura:

  • Inicialização : é aqui que você inicializa quaisquer variáveis ​​que usará no loop. Este código é executado apenas uma vez, no início do loop.
  • Condição : esta é uma expressão booleana avaliada antes de cada iteração do loop. Se a expressão for verdadeira, o loop continua; se for falso, o loop termina.
  • Incremento/Decremento : é aqui que você atualiza quaisquer variáveis ​​que estiver usando na expressão de condição. Este código é executado após cada iteração do loop.

Aqui está um exemplo de for loop que imprime os números de 0 a 4 no console:

for (let i= 0; i < 5; i++) {

ㅤㅤconsole.log(i);

}

Quando você inicializa a variável i com 0 na seção de inicialização e define a condição i < 5, o loop continuará executando enquanto i for menor que 5. Após cada iteração, a seção de incremento é executada, adicionando 1 a i.

É verdade que você pode omitir qualquer uma das três partes do for loop, dependendo das necessidades do seu código. Se você quiser fazer um loop indefinidamente, pode omitir a seção de condição, como neste exemplo:

for (let i= 0; ; i++) {

ㅤㅤconsole.log(i);

ㅤㅤ// Código a ser repetido indefinidamente

}

While{}

O while loop é um tipo de loop em JavaScript que permite executar um bloco de código repetidamente enquanto uma determinada condição for verdadeira. A sintaxe básica de um while loop é a seguinte:

while (condicao) {

ㅤㅤ// Bloco de código a ser executado repetidamente

}

Nesta estrutura:

  • A condição é avaliada antes de cada iteração do loop. Se for verdadeira, o bloco de código é executado; caso contrário, o loop é encerrado.

O while loop é útil quando você não sabe exatamente quantas vezes precisará repetir um bloco de código, mas sabe que deseja continuar enquanto uma determinada condição for verdadeira.

Aqui está um exemplo de whileloop que imprime os números de 0 a 4 no console:

let i = 0;

while (i < 5) {

ㅤㅤconsole.log(i);

ㅤㅤi++

}

No exemplo fornecido, a variável i é inicializada com 0 fora do loop. A condição i < 5 é avaliada antes de cada iteração, e o loop continua enquanto i for menor que 5. Após cada iteração, o bloco de código dentro do loop é executado, e o valor de i é atualizado adicionando 1 a ele.

Do...While{}

Uma consideração importante sobre os loops while é que se a condição for inicialmente falsa, o bloco de código dentro do loop não será executado. Isso pode ser uma fonte comum de bugs se não for tratado com cuidado. Para garantir que o bloco de código seja executado pelo menos uma vez, você pode usar um loop do...while, que sempre executa o bloco de código uma vez antes de verificar a condição novamente.

let i = 0;

do {

ㅤㅤconsole.log(i);

ㅤㅤi++

} while(i < 5);

No exemplo fornecido, o `do...while` loop executará o bloco de código uma vez, independentemente de a condição ser inicialmente falsa. Após a primeira execução do bloco de código, a condição é verificada. Se a condição for falsa, o loop terminará sem executar o bloco de código novamente.

Esta é uma distinção importante em relação ao loop `while` padrão: o `do...while` garante a execução do bloco de código pelo menos uma vez, enquanto um loop `while` padrão pode não executar o bloco de código se a condição for inicialmente falsa.

Essa característica torna o `do...while` loop útil em situações em que você precisa garantir que um bloco de código seja executado pelo menos uma vez, independentemente da condição inicial.
