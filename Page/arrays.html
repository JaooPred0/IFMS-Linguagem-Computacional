Arrays

Um array em JavaScript é uma estrutura de dados que permite armazenar múltiplos valores em uma única variável. Os arrays podem conter qualquer tipo de dados, incluindo números, strings, objetos e até mesmo outros arrays. Eles são bastante versáteis e amplamente utilizados em programação JavaScript.

Criando um Array

Existem várias maneiras de criar um array em JavaScript

  • Usando colchetes: Você pode criar um array simplesmente envolvendo uma lista de valores entre colchetes [ ]

    • const meuArray = [1, 2, 3, 4, 5];

  • Usando o construtor Array(): Você também pode usar o construtor Array() para criar um array.

    • const meuArray = new Array(1, 2, 3, 4, 5);

    Acessando Elementos do Array

    Os elementos em um array são acessados utilizando seus índices, começando em 0 para o primeiro elemento e indo até length - 1 para o último elemento.

    const meuArray = [10, 20, 30, 40, 50];

    console.log(meuArray[0]); // Saída: 10

    Funções Úteis de Arrays

    • push() e pop(): push() adiciona um elemento ao final do array, enquanto pop() remove e retorna o último elemento.

      • const meuArray = [1, 2, 3];

      meuArray.push(4); // meuArray agora é [1, 2, 3, 4]

      const elementoRemovido = meuArray.pop(); // elementoRemovido é 4, meuArray agora é [1, 2, 3]

    • shift() e unshift(): unshift() adiciona um elemento ao início do array, enquanto shift() remove e retorna o primeiro elemento.

      • const meuArray = [2, 3, 4];

      meuArray.unshift(1); // meuArray agora é [1, 2, 3, 4]

      const elementoRemovido = meuArray.shift(); // elementoRemovido é 4, meuArray agora é [2, 3, 4]

    • indexOf(): Retorna o índice do primeiro elemento correspondente ao valor especificado, ou -1 se o valor não for encontrado.

      • const meuArray = [10, 20, 30, 40, 50];

      const indeci = meuArray.indexOf(30); // índice é 2

    • slice(): Retorna uma cópia de parte do array, selecionada a partir do índice inicial até o índice final (ou até o final do array, se nenhum índice final for fornecido).

      • const meuArray = [1, 2, 3, 4, 5];

      const novoArray = meuArray.slice(1, 3); // novoArray é [2, 3]
