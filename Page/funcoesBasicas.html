Alert()

A `alert()` é uma função JavaScript integrada que exibe uma caixa de diálogo com uma mensagem ao usuário. A caixa de diálogo contém um botão OK e o usuário pode clicar nele para fechar a caixa de diálogo.

Aqui está um exemplo de como usar a `alert()`:

alert('Hello, world!');

Ao executar este código, aparecerá uma caixa de diálogo com a mensagem "Olá, mundo!" e um botão OK. O usuário pode clicar no botão OK para fechar a caixa de diálogo e continuar executando o restante do código.

A `alert()` pode ser útil em determinadas situações, como:

  • Mostrar
  • Depurando código exibindo o valor de uma variável ou expressão

Porém, é importante usar a `alert()` com cautela, pois ela pode interromper o fluxo de trabalho do usuário e ser irritante se usada excessivamente. Geralmente é melhor usar outros métodos de feedback do usuário, como modificar o conteúdo da página ou usar um sistema de notificação mais sutil, se possível.

Aqui está um exemplo de uso `alert()` para depuração:

let x = 5;

let y = 10;

let resultado = x + y;

alert('O resultado é:', resultado);

Neste exemplo, estamos usando `alert()` para exibir o resultado da adição de dois números. Isso pode ser útil para depuração, pois nos permite verificar rapidamente o valor de uma variável ou expressão sem usar uma ferramenta de depuração mais complexa.

Alert() ou window.alert()

Em JavaScript, a função `alert()` é usada para exibir uma caixa de diálogo com uma mensagem para o usuário. Esta função é parte do objeto global window nos navegadores da web. Isso significa que você pode chamar `alert()` como uma função independente ou como um método do objeto `window`, pois `window` é o escopo global padrão no navegador.

alert('Hello world');

window.alert('Hello world');

O `window` é o objeto de nível superior no ambiente do navegador e contém diversas funções e propriedades integradas que estão disponíveis para o código JavaScript em execução no navegador.

Quando se chama `alert()` sem especificar o objeto `window`, o JavaScript assume que se está referindo ao método `alert()` do objeto `window`. Isso ocorre porque `alert()` é uma função global e as funções globais são automaticamente adicionadas como métodos do objeto `window`.

Prompt() Geral

prompt()

A `prompt()` é uma função integrada em JavaScript que exibe uma caixa de diálogo com uma mensagem e um campo de entrada para o usuário inserir um valor. A função retorna um valor de string da entrada do usuário. Aqui está um exemplo:

const userInput = prompt('Digite alguma coisa:');

console.log(userInput);

Neste exemplo, se o usuário inserir '25' na caixa de diálogo, `userInput` será uma string com o valor '25'.

+prompt

A `+prompt()` sintaxe é uma forma abreviada de converter a entrada do usuário em um número. O operador unário mais `+` é usado para converter uma string em um número. Aqui está um exemplo:

const userInput = +prompt('Digite alguma coisa:');

console.log(userInput);

Neste exemplo, se o usuário inserir '25' na caixa de diálogo, `userInput` será um `número` com o valor 25.

Number(prompt())

A `Number(prompt())` sintaxe é semelhante a `+prompt()` , mas usa a `Number()` em vez do operador unário mais. A `Number()` converte um valor em um número. Aqui está um exemplo:

const userInput = Number(prompt('Digite alguma coisa:'));

console.log(userInput);

Neste exemplo, se o usuário inserir '25' na caixa de diálogo, `userInput` será um `número` com o valor 25.

parseFloat(prompt())

A `parseFloat(prompt())` sintaxe é semelhante a `Number(prompt())`, mas converte apenas a parte numérica da entrada do usuário. Se o usuário inserir uma string que contenha caracteres não numéricos, ` parseFloat()` retornará um número até o último caractere numérico e ignorará o restante. Aqui está um exemplo:

const userInput = parseFloat(prompt('Digite alguma coisa:'));

console.log(userInput);

Neste exemplo, se o usuário inserir '123.45abc' na caixa de diálogo, `userInput` será um número com o valor 123,45.

Em resumo, `prompt()` retorna um valor de string da entrada do usuário, enquanto `+prompt()`, `Number(prompt())` e parseFloat(prompt()) converte a entrada do usuário em um número. A principal diferença entre `+prompt()`, `Number(prompt())` e `parseFloat(prompt())` é como eles lidam com caracteres não numéricos na entrada do usuário. `+prompt()` e `Number(prompt())` retornará NaNse a entrada do usuário contiver caracteres não numéricos, enquanto `parseFloat(prompt())` retornará um número até o último caractere numérico e ignorará o restante.

Confirm()

Certamente! A confirm() é uma função integrada em JavaScript que exibe uma caixa de diálogo com uma mensagem e dois botões: OK e Cancelar. A função retorna um valor booleano com base na resposta do usuário.

Aqui está um exemplo de como usar a confirm():

const isUserSure = confirm('Deseja deletar esse arquivo?');

console.log(isUserSure);

Neste exemplo, se o usuário clicar no botão OK, `isUserSure` será verdadeiro. Se o usuário clicar no botão Cancelar, `isUserSure` será falso.

A função `confirm()` pode ser útil em situações em que você deseja solicitar confirmação ao usuário antes de realizar uma ação destrutiva, como excluir um registro ou fechar uma janela.

É importante observar que a função `confirm()` bloqueia a execução do código JavaScript até que o usuário responda à caixa de diálogo. Isso significa que se você tiver outro código JavaScript que precise ser executado após a função `confirm()`, ele não será executado até que o usuário responda à caixa de diálogo.

Document.white()

document.write()é um método em JavaScript que grava uma string de texto no documento atual. Ele pode ser usado para adicionar ou modificar conteúdo em uma página da web.

Aqui está um exemplo de como usar document.write():

document.white('< h1> Hello, World! < / h1>);

Neste exemplo, o document.write() método adiciona um elemento com o texto “Hello, World!” ao documento atual.

Conclusão

A função `alert()` é uma ferramenta crucial em JavaScript para exibir mensagens em uma caixa de diálogo para os usuários. Quando você chama `alert()`, uma caixa de diálogo simples com uma mensagem e um botão "OK" é exibida, permitindo que os usuários interajam com ela. Isso é útil para fornecer feedback imediato ou solicitar a atenção do usuário em determinados pontos de sua aplicação.

É importante usar a função `alert()` com parcimônia, pois ela pode interromper o fluxo de trabalho do usuário e ser considerada irritante se usada em excesso. Geralmente, é preferível utilizar métodos mais sutis de feedback ao usuário, como modificar o conteúdo da página ou utilizar sistemas de notificação mais discretos, quando possível.

Além disso, é relevante destacar que a função `alert()` é parte do objeto global `window` em navegadores da web. Isso significa que você pode chamá-la tanto como uma função independente quanto como um método do objeto `window`, sendo `window.alert()` uma forma mais explícita de fazer isso.

Em resumo, a função `alert()` é uma ferramenta poderosa em JavaScript para interagir com os usuários, mas deve ser usada com cuidado para não prejudicar a experiência do usuário. Optar por métodos de feedback mais sutis e menos invasivos sempre que possível pode levar a uma melhor experiência do usuário em sua aplicação.
