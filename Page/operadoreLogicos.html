Operadores Lógicos

Em JavaScript, os operadores lógicos são usados para combinar expressões booleanas. Eles são usados para executar operações lógicas como AND, OR e NOT.

  • Operador AND (&&): Este operador retorna true se ambas as expressões forem verdadeiras. Caso contrário, retorna false.

    • let a = 5;

    let b = 10;

    console.log(a < b && a > 0);

  • Operador OR (||): Este operador retorna true se pelo menos uma das expressões for verdadeira. Caso contrário, retorna false.

    • let a = 5;

    let b = 10;

    console.log(a > b || a > 0);

  • Operador NOT (!): Este operador retorna o valor oposto de uma expressão booleana. Se a expressão for verdadeira, o operador NOT retornará false. Se a expressão for falsa, o operador NOT retornará true.

    • let a = 5;

    console.log(!(a > 100));

  • Operador AND curto-circuito (&&=): Este operador é uma combinação do operador AND e do operador de atribuição. Ele retorna o valor da segunda expressão se a primeira expressão for verdadeira. Caso contrário, retorna o valor da primeira expressão.

    • let a = 5;

    let b = 10;

    a &&= b > 0;

    console.log(a);

  • Operador OR curto-circuito (||=): Este operador é uma combinação do operador OR e do operador de atribuição. Ele retorna o valor da segunda expressão se a primeira expressão for falsa. Caso contrário, retorna o valor da primeira expressão.

    • let a = 5;

    let b = 10;

    a ||= b > 0;

    console.log(a);
